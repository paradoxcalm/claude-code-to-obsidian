# Логи сессий Claude Code

Здесь автоматически сохраняются логи сессий работы с Claude Code.

## Поиск
- По дате: `YYYY-MM-DD` в названии файла
- По проекту: тег `#проект-название`
- Dataview: `TABLE file.name, file.cday FROM "sessions" SORT file.cday DESC`

## Структура лога
Каждый файл содержит: что сделано, изменённые файлы, ключевые решения, TODO.

## Файлы с точкой (скрытые)
- `.tool-log-YYYY-MM-DD.txt` — счётчик вызовов инструментов (технический)
- `.logged-SESSION_ID` — маркеры записанных сессий (технический)
