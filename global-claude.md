# Глобальные инструкции Claude Code

## Obsidian Vault — автологирование сессий
Путь к vault: `__VAULT_PATH__`

### Система хуков:
1. **PostToolUse** → считает вызовы инструментов
2. **Stop** → если сессия существенная (5+ tool calls) и лог не записан — впрыскивает `[АВТОЛОГ]` напоминание
3. **SessionEnd** → создаёт заготовку лога при выходе

### При получении напоминания `[АВТОЛОГ]`:
Создай файл в `__VAULT_PATH__/sessions/` :

**Имя:** `YYYY-MM-DD_HH-MM_название-проекта.md`

**Содержимое:**
```markdown
# Сессия: [Краткое описание]

**Дата:** YYYY-MM-DD HH:MM
**Проект:** [имя]
**Директория:** [путь]

## Что сделано
- [пункт]

## Изменённые файлы
- `путь/файл` — что изменено

## Ключевые решения
- [решение] — [почему]

## TODO
- [ ] [задача]

#сессия #[проект] #[тема]
```

### После записи:
```bash
touch "__VAULT_PATH__/sessions/.logged-SESSION_ID"
```

### Правила:
- Пиши на русском
- Будь конкретным: не "изменил файл", а "добавил валидацию в форму"
- Короткие сессии (< 5 tool calls) — лог не нужен
