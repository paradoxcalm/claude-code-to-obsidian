# Claude Code Session Logs

Session logs from working with Claude Code are automatically saved here.

## Search
- By date: `YYYY-MM-DD` in the file name
- By project: tag `#project-name`
- Dataview: `TABLE file.name, file.cday FROM "sessions" SORT file.cday DESC`

## Log Structure
Each file contains: what was done, changed files, key decisions, TODO.

## Dot files (hidden)
- `.tool-log-YYYY-MM-DD.txt` — tool call counter (technical)
- `.logged-SESSION_ID` — markers for recorded sessions (technical)
